# twitterClone
 
A Twitter Clone with Node.js, Socket.IO and MongoDB

TechStack:
Node JS
Socket IO
JavaScript
MongoDB
Mongoose
HTML
CSS


Functionalities:
    Signup/Login
    
    Tweets

    Likes, comments and retweets

    Profile pages

    Following and followers support

    Instant messaging in real time

    Group chat support

    Profile pictures

    Cover photos

    and much more!
    
    
    How to run the project :
    1.clone the app
    2. run npm install in vs code or cmd (where you clone the project)
    3.npm start
    4. go to browser and type localhost:3003 
    5.that's it
